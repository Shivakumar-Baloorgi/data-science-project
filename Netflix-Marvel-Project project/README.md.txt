# Netflix Marvel Movies Data Science Project

## Objective
Analyze the most watched Marvel movies on Netflix using Python and data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Kaggle Notebook

## Features
- Data Analysis
- Bar Charts
- Correlation Analysis
- IMDb Ratings Comparison

## Dataset
Contains Marvel movie ratings, runtime, and estimated views.
